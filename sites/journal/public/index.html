Journal

Journal is the home for durable markdown notes, implementation logs, technical references, and reusable AI context.

Start with the section directories below. The content model and visual theme are intentionally minimal here so later tickets can add structure without fighting the scaffold.

Sections

  • Ideas

    Loose concepts, opportunities, and speculative notes live here.

  • Journals

    Daily notes, implementation logs, and working journal entries live here.

  • Planner

    Planning pages, checklists, and execution notes live here.

  • References

    Reusable references, precedents, and evergreen notes live here.

  • Technical
